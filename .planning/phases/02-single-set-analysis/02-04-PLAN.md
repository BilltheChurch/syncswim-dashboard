    - Load data: `imu_df = load_imu(selected["path"])`, `vision_df = load_vision(selected["path"])`

    **IMU waveform section (if imu_df not empty):**
    - Compute derived signals:
      - `accel_mag = np.sqrt(imu_df["ax"]**2 + imu_df["ay"]**2 + imu_df["az"]**2).values`
      - `gyro_mag = np.sqrt(imu_df["gx"]**2 + imu_df["gy"]**2 + imu_df["gz"]**2).values`
      - `tilt = calc_imu_tilt(imu_df.to_dict("records"))`, `tilt_smooth = smooth(tilt, 15)`
      - `time_sec = (imu_df["timestamp_local"] - imu_df["timestamp_local"].iloc[0]).values` (convert epoch to relative seconds)
    - `st.subheader("IMU 波形")` + plotly_chart(build_imu_waveform(...))
    - If imu_df empty: `st.warning("该训练组无 IMU 数据。")`

    **Fusion chart section (if BOTH imu_df and vision_df not empty):**
    - `st.divider()` + `st.subheader("传感器融合")`
    - **Timestamp alignment procedure** (both DataFrames use "timestamp_local" column, both in epoch seconds float):
      1. Compute relative time for each: `imu_t = (imu_df["timestamp_local"] - imu_df["timestamp_local"].iloc[0]).values` and `vis_t = (vision_df["timestamp_local"] - vision_df["timestamp_local"].iloc[0]).values`
      2. Extract vision angles: `vis_angles = vision_df["angle_deg"].values`
      3. Extract IMU tilt (already computed above as tilt_smooth)
      4. Resample IMU tilt onto vision timestamps using np.interp: `imu_tilt_resampled = np.interp(vis_t, imu_t, tilt_smooth)`
      5. Handle edge case: if either DataFrame is empty (len==0), skip fusion chart and show `st.info("融合图表需要同时有 IMU 和视觉数据。")`
    - `fusion_fig, corr = build_fusion_chart(vis_t, vis_angles, imu_tilt_resampled)`
    - `st.plotly_chart(fusion_fig, use_container_width=True)`
    - Correlation badge: `if corr is not None:` determine color (green if abs>=0.7, yellow if abs>=0.4, red otherwise), show `st.metric("相关系数", f"r = {corr:.3f}")`

    **If vision_df not empty but no IMU:** show vision-only angle info
    **If neither has data:** `st.error("传感器数据不可用。")`
