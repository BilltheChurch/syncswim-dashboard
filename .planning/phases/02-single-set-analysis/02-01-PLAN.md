    **TDD GREEN phase:** Implement the three modules:

    **dashboard/core/scoring.py:**
    - `@dataclass MetricResult`: fields name (str), value (float), unit (str), deduction (float), zone (str), max_value (float)
    - `@dataclass SetReport`: fields metrics (list[MetricResult]), overall_score (float), phases (list[dict]), correlation (float | None)
    - `compute_deduction(value: float, config: dict) -> tuple[float, str]`: Apply FINA thresholds. If value < config["fina"]["clean_threshold_deg"] return (config["fina"]["clean_deduction"], "clean"). If value < config["fina"]["minor_deduction_deg"] return (config["fina"]["minor_deduction"], "minor"). Else return (config["fina"]["major_deduction"], "major").
    - `compute_leg_deviation(imu_df: pd.DataFrame) -> float`: Call calc_imu_tilt(imu_df.to_dict("records")), then smooth(tilt, window=15), return float(np.mean(np.abs(tilt_smooth - 90.0)))
    - `compute_leg_height_index(vision_df: pd.DataFrame) -> float`: Extract left_hip (lm 23), left_ankle (lm 27) Y coords from landmarks if available, else use angle_deg column mean as proxy. Ratio = (ankle_y - water_line) / (hip_y - ankle_y). For Phase 2 MVP without landmarks.csv, return mean angle_deg as proxy metric.
    - `compute_shoulder_knee_alignment(vision_df: pd.DataFrame) -> float`: Extract shoulder (lm 12), hip (lm 24), knee (lm 26) as (x,y) tuples, call calc_angle(shoulder, hip, knee). For Phase 2 MVP without landmarks.csv, return 180.0 - mean(angle_deg) as proxy.
    - `compute_smoothness(imu_df: pd.DataFrame) -> float`: gyro_mag = sqrt(gx^2+gy^2+gz^2), smooth(gyro_mag, 5), return float(np.mean(np.abs(np.gradient(smoothed))))
    - `compute_stability(imu_df: pd.DataFrame, phase_bounds: tuple[float, float]) -> float`: Filter IMU rows where timestamp_local is within phase_bounds, call calc_imu_tilt on filtered data, return float(np.std(tilt))
    - `compute_set_report(imu_df: pd.DataFrame, vision_df: pd.DataFrame, config: dict) -> SetReport`: Orchestrate all metrics. Detect phases via phase_detect.detect_phases(imu_df). Compute correlation via np.corrcoef if both DataFrames non-empty. Return SetReport with overall_score = 10.0 - sum(deductions). Handle partial data: if imu_df empty skip IMU metrics, if vision_df empty skip vision metrics.

    **dashboard/core/phase_detect.py:**
    - `butterworth_filter(data: np.ndarray, cutoff=10.0, fs=72.5, order=4) -> np.ndarray`: Butterworth low-pass. Check len(data) < 3 * max(len(a), len(b)) and return data unfiltered as fallback.
    - `detect_phases(imu_df: pd.DataFrame, n_phases=3) -> list[dict]`: Compute accel_mag = sqrt(ax^2+ay^2+az^2). Apply butterworth_filter. find_peaks(filtered, prominence=0.3, distance=int(72.5)). If >= 2 peaks, use top-2 prominence peaks as boundaries. Else fallback to equal thirds. Return list of 3 dicts with keys: name ("准备"/"动作"/"恢复"), start (float), end (float), zone_color ("#09AB3B"/"#FACA2B"/"#09AB3B"). Note: zone_color defaults are green/yellow/green; actual quality coloring is computed later based on per-phase metric values.

    **dashboard/core/metrics.py:**
    - `compute_all_metrics(set_dir: str) -> SetReport | None`: Load IMU and vision DataFrames via load_imu/load_vision, load config via load_config, call compute_set_report. Return None if both DataFrames empty. This is the single entry point for the training page.
