① 怎么开始

步骤动作
1. 启动双击桌面 start-cvat.command → 等 30 秒
2. 浏览器会自动打开 CVAT；如果没开 → 手动访问 http://localhost:8080
3. 登录用户名 emily + 你的密码（浏览器记住了）
4. 进任务Project syncswim-detector-phase-a → Task → Job #1

② 画 bbox 的核心规则（必读）

✅ 标什么

  • 水里的运动员（哪怕只露脚 / 头）
  • bbox 包到水下能看到的身影，不是水面
  • 多个运动员 = 每人一个独立框
  • 同帧、同距离的运动员 bbox 风格统一

❌ 不要标

  • 教练 / 观众 / 路人
  • 水面倒影、影子
  • 完全被挡看不到的人（连水花都没）
  • 不要 "猜位置" — 看不见的不要框

③ CVAT 操作要点

操作怎么做
新建 bbox左侧工具栏点中 矩形 ▢ 图标 → 按 N → 弹窗选 Shape + By 2 Points → 拖矩形
下一帧F
上一帧D
删除选中 bboxDelete
保存自动每 30 秒保存；强制保存 Ctrl+S
撤销Ctrl+Z

绝对不要选 Track（带帧导航箭头的）— 我们要的是 Shape。

④ 边界 case 速查

场景怎么标
运动员只露腿，水下能看到身影bbox 从脚尖一直拉到水下身影末端
两人重叠 60% 以上照标，bbox 重叠 OK
完全被挡看不到跳过，不要猜位置
只露一只手 / 一个脚趾框那一小部分（哪怕 20×30 px）
水面只有水花，看不出是谁跳过
远近运动员大小差很大正常 — 距离造成的，不算不一致
